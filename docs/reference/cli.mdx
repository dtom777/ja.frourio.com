## Frourio {#frourio}

<img src="/images/logo.svg" width="144px" height="40px" alt="frourio" />

### オプション {#frourio-options}

#### `[--version/-v]` {#frourio-version}

Frourio のバージョンを表示します。

#### `[--watch/-w]` {#frourio-watch}

ウォッチモードを有効にします。
API ファイルの変更に伴って `$server.ts` を再生成します。

#### `[--project/-p <project dir>]` {#frourio-project}

デフォルト: `.`

Frourio はこのパスを用いて **tsconfig** を参照します。

## Create Frourio App {#cfa}

### 引数 {#cfa-arguments}

#### `[<dir>]` {#cfa-dir}

デフォルト: `my-frourio-app`

### オプション {#cfa-options}

#### `[-v]` {#cfa-version}

create-frourio-app のバージョンを表示します。

#### `[--port/-p <port>]` {#cfa-port}

デフォルト: `3000`

#### `[--host <hostname>]` {#cfa-host}

デフォルト: `localhost`

#### `[--answers <answers json>]` {#cfa-answers}

このオプションが指定されると、cfa は GUI モードの代わりに CUI モードで起動します。

JSON ジェネレーター: [CUI によるセットアップ](/docs/reference/cfa/cui)

#### `[--help/-h]` {#cfa-help}

ヘルプを表示します。

## Pathpida {#pathpida}

<img
  src="https://aspida.github.io/pathpida/logos/svg/black.svg"
  width="200px"
  height="40px"
  alt="pathpida"
/>

詳細情報: [aspida/pathpida](https://github.com/aspida/pathpida#CLI-options)

### オプション {#pathpida-options}

#### `[--version/-v]` {#pathpida-version}

Pathpida のバージョンを表示します。

#### `[--watch/-w]` {#pathpida-watch}

ウォッチモードを有効にします。
ページファイルや静的ファイルの変更に伴って `$path.ts` を再生成します。

#### `[--enableStatic/-s]` {#pathpida-enable-static}

静的ファイルのパスを `$path.ts` に生成します。

#### `[--ignorePath/-p <ignore pattern>]` {#pathpida-ignore-path}

無視するファイルのパターンを指定します。

#### `[--output/-o <output dir>]` {#pathpida-output}

デフォルト:

- Next.js: `utils` / `lib`
- Nuxt.js: `plugins`
- その他: `node_modules`

`$path.ts` を出力するディレクトリを指定します。
