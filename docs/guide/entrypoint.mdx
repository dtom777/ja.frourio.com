id: entrypoint
title: Customize Entrypoint
---

The instance of fastify or express is created in `server/service/app.ts` and is started in `server/entrypoint/index.ts`.

Change `$/service/app.ts` if you need to access the server instance to register plugins, and `$/entrypoint/index.ts` if you need to change the IP address or port to listen.
