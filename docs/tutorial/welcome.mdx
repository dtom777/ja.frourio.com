Frourio エコシステムへようこそ。

このチュートリアルでは、create-react-app と Fastify を用いた基本的なクライアント + サーバー構造に aspida と frourio を取り入れることで、それらの基本的な機能を学ぶことができます。

:::info
このチュートリアルでは yarn を使用しますが、npm など他のパッケージマネージャーを使うこともできます。
:::
