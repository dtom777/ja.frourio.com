Setup with GUI

  • npx
  • npm
  • yarn
npx create-frourio-app

Once the above command is installed, the browser will automatically open at http://localhost:3000

http://localhost:3000

Fill in the blanks and press the Create button at the bottom to start installation again and change your browser to Next.js or Nuxt.js.

  • Next.js
  • Nuxt.js
http://localhost:3000

note

Setup may fail in some WSL environments. If so, set it up with CUI.
